Installation

  1. Clone the repository
  2. Open the folder in Cursor IDE
  3. Install the companion MCP server: <code>npm install -g @tmhs/steam-mcp</code>
  4. Add your Steam API key to the MCP config
  5. Start using the AI skills and rules
